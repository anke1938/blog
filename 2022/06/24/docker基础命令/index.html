    基础命令

    手动保存/装载image
    离线安装镜像，在联网主机上：
    $ docker pull hello-world
    $ docker save –output hello-world.tar {your image name or ID}

    在离线主机上：

    $docker load –input hello-world.tar

    手动打包镜像：

    docker image build -t flask_api .
